It's been a busy week across the Canvas Computing projects.

The headline is a brand-new repo: [`malwi-benchmark`](https://github.com/canvascomputing/malwi-benchmark), published yesterday. It's a benchmark for evaluating how well agentic systems can detect supply-chain attacks — essentially a test suite for `malwi` itself. Building a proper benchmark feels like a maturity milestone for the project; it means we can start measuring progress objectively rather than just eyeballing results.

Also this week, the `agent` Rust crate was renamed and refocused into [`agentwerk`](https://github.com/canvascomputing/agentwerk), which had a flurry of commits landing today. The new name and positioning ("gives *any* application agentic capabilities") feels cleaner — it's less about being a standalone framework and more about being a composable building block.

Over in [`Kyri-Cou/cryptography`](https://github.com/Kyri-Cou/cryptography), the companion site for the YouTube cryptography course is still getting updates. Commits landed today, so new lessons or interactive tools are likely in the pipeline.

On the personal projects side, `fasting-app` got a push early in the week, and this site itself had its first full automated weekly cycle run — the projects page now stays in sync with whatever's happening across all the repos.
